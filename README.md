# Timecode

Timecode is a free, open-source, self-hosted coding activity tracker.

V1 scope:
- Single machine only (no sync)
- VS Code extension captures coding activity
- Local SQLite database stores events and daily summaries
- Local web dashboard visualizes stats
- PNG export for custom date ranges

All components run on the client machine.
